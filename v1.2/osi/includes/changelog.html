OSI Color Reference Manager
Version History & Release Notes

Current Version
v1.2 - Released: February 20, 2014
  • Added ability to sort by column when viewing color searches. Column sorting is available on keyword searches, advanced searches, and browsing by manufacturer. This functionality extends to HTML exported files as well.
  • Pages with large vertical scrolling areas will now display a "Back to Top" link in the lower right hand corner of the screen. The link will show as you scroll deeper into a page.
  • Replaced previously forced compatibility mode of Internet Explorer 8 with Internet Explorer 9.
  • Reduced file sizes on some imported Javascript files to reduce bandwidth.
  • Minor UI tweaks.
  • Minor bug fixes.
Previous Versions
v1.1.1 - Released: July 18, 2013
  • Fixed a bug introduced in v1.1 that prevented the color editor from being displayed to any user with the "Edit Color" permission setting.
  • Fixed regular expression handling of searches using a "/" in the query pattern.
v1.1 - Released: July 10, 2013
  • Added export functionality for .CSV and .HTML files when browsing colors by manufacturer. This feature was previously available but could only be achieved using the advanced search.
  • Implemented a custom sorting and scoring solution for search results based on keyword relevancy. In certain cases some searches that would return many results (e.g. "white") would have records sorted in an arbitrary manner. The new sorting technique will attempt to place more relevant results near the top of the list when displaying on screen.
  • Modified the search algorithm to be more inclusive with short keywords. Certain keywords containing three characters or less would not always return records that should have been included with the results.
  • Fixed a discrepancy in the code generated in HTML exports causing the document to fail passing W3C XHTML 1.0 Transitional validation.
  • Indentified and fixed minor bugs.
v1.0.2 - Released: May 17, 2013
  • Reconfigured the internal session settings to prevent the hosting provider from automatically logging users out of the software after very short periods of time.
v1.0.1 - Released: May 15, 2013
  • Fixed a bug in the batch color processor where a new manufacturer would be created but colors associated to it would not.
v1.0 - Released: May 13, 2013
  • OSI Color Reference Manager created.
  • Internal beta testing completed and software compiled into production version.
  • Software deployed and integrated with the CSI website.
